# Random Quote

This project generate a random quotes.
If you like to share you can copy the quote.

## Run

Install the dependencies

```bash
  pnpm install
```

To run this project

```bash
  pnpm run dev
```

```bash
  npm run dev
```

## API Reference

#### Get random quote

```http
  GET https://api.quotable.io/random
```
